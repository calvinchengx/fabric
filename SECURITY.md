# Security

## Reporting potential security issues

If you believe you have found a security vulnerability in this project, **do not** open a public issue describing it.

Please use **[GitHub’s private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)** for this repository: open the **Security** tab, then **Report a vulnerability**. That keeps details private while we triage and fix the issue.

We will work with you to confirm the report and coordinate a fix and disclosure timeline.
